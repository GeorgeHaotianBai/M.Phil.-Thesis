\section{Supplementary Materials: Replication Package} \label{app:supplementary}

The supplementary materials accompanying this paper are organised into two
top-level folders, \texttt{code/} and \texttt{data/}, which together
regenerate the third folder, \texttt{output/}, end-to-end. The
\texttt{output/} folder contains every figure cited in the main text and
appendix and the auto-generated \texttt{tabularx} bodies used by Tables 2,
3, and A.1. All scripts read only from paths inside the project tree;
there are no external-folder dependencies. This appendix documents the
provenance of each input data file. A separate \texttt{README.md} in the
replication package details the repository layout, build sequence,
software requirements, and the script-to-figure mapping.

\subsection{Analysis-ready CSVs and their provenance}

The folder \texttt{data/for\_analysis/} contains one CSV per figure (or per
panel where panels need different inputs). The naming convention is
\texttt{figNN\_<descriptor>.csv}, so a reader can locate every input a
script will read by inspecting the script's prefix. Provenance for each raw
input is summarised below.

\subsection{Software requirements}

\begin{itemize}[leftmargin=*, itemsep=0.2em]
\item \textbf{Python 3.10+} with \texttt{numpy}, \texttt{pandas},
      \texttt{matplotlib}, \texttt{statsmodels}, \texttt{linearmodels},
      \texttt{geopandas} (Fig.~6 maps only), \texttt{Pillow}, and
      \texttt{anthropic} (only for re-running the Haiku classifier on
      raw speech text; the classifier output is shipped as the
      \texttt{fig01\_classified\_*.csv} files, so no API key is required
      to reproduce Fig.~3).
\item \textbf{Stata MP 17+} for the event-study pipeline, with the
      following user-installed packages (vendored in \texttt{ado/plus/}
      under the Stata folder for portability):
      \texttt{reghdfe}, \texttt{ftools}, \texttt{stackedev},
      \texttt{did2s}, \texttt{event\_plot}, \texttt{fect}.
\item \textbf{LaTeX} (TeX Live 2024+) with the standard AER toolchain:
      \texttt{natbib}, \texttt{floatrow}, \texttt{tabularx},
      \texttt{pdflscape}, \texttt{subfigure}, \texttt{cleveref},
      \texttt{hyperref}.
\end{itemize}
